# Personas e Público-Alvo

## 🎯 Visão Geral

FiscalOS possui dois públicos principais:
1. **Primário:** Contadores e escritórios de contabilidade
2. **Secundário:** Empresários autônomos que fazem sua própria gestão

---

## 👤 Persona Primária: Carlos - O Contador Solo

### Dados Demográficos

| Atributo | Descrição |
|----------|-----------|
| **Nome** | Carlos Silva |
| **Idade** | 38 anos |
| **Profissão** | Contador autônomo |
| **Localização** | São Paulo, SP |
| **Experiência** | 15 anos na área |
| **Clientes** | 60-80 empresas |
| **Faturamento** | R$ 15.000 - R$ 25.000/mês |

### Perfil Profissional

- Trabalha sozinho ou com 1 assistente
- Atende empresas de diversos portes (MEI a média)
- Usa Excel e sistemas contábeis tradicionais
- Muitas demandas, pouco tempo

### Comportamento

- 🔴 **Ocupado:** Agenda sempre cheia
- 🔴 **Sobrecarregado:** Trabalha finais de semana
- 🔴 **Manual:** Ainda faz muitos cálculos no Excel
- 🟢 **Interessado:** Busca modernizar o escritório
- 🟢 **Receptivo:** Aberto a novas ferramentas

### Dores

1. **"Planejamento fiscal me consome horas"**
   - Cada cliente leva 4-6 horas
   - Precisa refazer quando dados mudam
   - Muitos cálculos repetitivos

2. **"Perco muito tempo com planilhas"**
   - Excel trava com muitos dados
   - Cálculos manuais suscetíveis a erro
   - Dificuldade em padronizar

3. **"Não tenho tempo para formar preço direito"**
   - Cobra barato pelo serviço
   - Não consegue atender mais clientes
   - Risco de burnout

### Necessidades

| Necessidade | Prioridade |
|-------------|------------|
| Automatizar planejamento fiscal | 🔴 Alta |
| Gerar relatórios profissionais | 🔴 Alta |
| Atender mais clientes | 🟡 Média |
| Modernizar o escritório | 🟡 Média |
| Reduzir erros | 🔴 Alta |

### Objeções

- *"Será que vale o investimento?"*
- *"Tenho que aprender outro sistema?"*
- *"Meus dados ficam seguros?"*
- *"Funciona para todos os regimes?"*

### Gatilhos de Compra

- Cliente pedindo planejamento fiscal
- Fim do ano (planejamento para o próximo ano)
- Mudança de regime tributário
- Insatisfação com Excel/processos manuais

### Como FiscalOS Ajuda Carlos

| Problema | Solução FiscalOS |
|----------|------------------|
| Demora no planejamento | Automatização total em minutos |
| Relatórios manuais | Geração automática de PDF |
| Erros de cálculo | Motor de cálculo confiável |
| Poucos clientes atendidos | Produtividade 5-10x maior |

## 👥 Persona Primária: Fernanda - O Escritório de Contabilidade

### Dados Demográficos

| Atributo | Descrição |
|----------|-----------|
| **Nome** | Fernanda Oliveira |
| **Idade** | 45 anos |
| **Cargo** | Sócia-proprietária |
| **Empresa** | Oliveira Contabilidade |
| **Equipe** | 8 funcionários |
| **Clientes** | 300+ empresas |
| **Faturamento** | R$ 150.000 - R$ 250.000/mês |

### Perfil Profissional

- Escritório consolidado no mercado
- Equipe especializada por área
- Processos ainda muito manuais
- Busca escalabilidade

### Comportamento

- 🟢 **Estratégica:** Pensamento em crescimento
- 🟢 **Investidora:** Disposta a pagar por soluções
- 🔴 **Desconectada:** Perde tempo em operação
- 🟢 **Líder:** Equipe depende dela para decisões
- 🟡 **Em transição:** Modernizando processos

### Dores

1. **"Minha equipe gasta muito tempo em planejamento"**
   - Funcionários sobrecarregados
   - Demora na entrega para clientes
   - Improdutividade

2. **"Não consigo escalar o serviço de planejamento"**
   - Limitado pela capacidade da equipe
   - Não consegue atender todos clientes
   - Perde receita potencial

3. **"Falta padronização entre funcionários"**
   - Cada um faz do seu jeito
   - Relatórios diferentes
   - Qualidade inconsistente

### Necessidades

| Necessidade | Prioridade |
|-------------|------------|
| Escalar atendimento | 🔴 Alta |
| Padronizar processos | 🔴 Alta |
| Treinar equipe rápido | 🟡 Média |
| Aumentar margem de lucro | 🔴 Alta |
| Dashboard gerencial | 🟢 Baixa |

### Objeções

- *"Minha equipe vai aceitar usar?"*
- *"Vale o investimento para o escritório?"*
- *"Consigo integrar com meus sistemas?"*
- *"Suporte é bom?"*

### Gatilhos de Compra

- Demanda crescendo de planejamentos
- Funcionários reclamando de sobrecarga
- Concorrência mais moderna
- Necessidade de escalar

### Como FiscalOS Ajuda Fernanda

| Problema | Solução FiscalOS |
|----------|------------------|
| Equipe sobrecarregada | Automatização libera tempo |
| Não escala | Multi-usuário, ilimitados planejamentos |
| Sem padronização | Relatórios consistentes |
| Margem baixa | Mais valor em menos tempo |

## 🎯 Público-Alvo para MVP

### Focus Inicial

**Contadores solo e pequenos escritórios (até 5 pessoas)**

### Por quê?

| Razão | Explicação |
|-------|------------|
| ✅ Decisão rápida | Não precisa aprovar com comitê |
| ✅ Dor latente | Sofrem mais com falta de tempo |
| ✅ Facilidade de adoção | Menos burocracia interna |
| ✅ Volume de mercado | Maioria dos contadores são solo |
| ✅ Feedback direto | Falam direto com o desenvolvedor |

## ✅ Conclusão

FiscalOS deve focar inicialmente em **contadores solo e pequenos escritórios**, que sentem a dor da falta de tempo e buscam modernização. A persona principal é "Carlos", o contador solo que precisa de agilidade e profissionalismo.
